2 fold Symmetry

A simple representation of a 2 fold is a straight line representing the axis. Objects are rotated around this axis for 180° to create a symmetry related object. If the first object is represented above the page then it will be given a "+" notation and similarly below the page will have a "-" notation.



If we rotate this by 90° then we can visualise the objects looking down the 2 fold. This time both objects will be at the same level. The axis is represented by the lozenge indicating a 2 fold.
